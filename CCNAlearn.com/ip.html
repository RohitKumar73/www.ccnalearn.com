Internet protocol(IP)

Internet protocol(IP) is a host-to-host network layer delivery protocol designed for the internet.Its is a connectionless datagram protocol with no reliability assurance .Its is unrelianle as is does not provide and error contro or flow contol mechanism.

ip provides the following services:
  1. Address: IP headers contain 32bit address that identify the sending and receiving hosts
  2. Fragmentation: IP packets may be split or fragmented into smaller packets.
  3. Packet Timeout: Each IP packet contains a Time-to-Live field, which is decremented every time a router bandles ths packet . if RRl reaches zero the packet is discarded priventing packets from runnning in circlis foreer and flooding a network.
  4. Type of Service: IP supports traffic prioritization by allowing packets to be labeled

IP Addressing

Classic TCP/IP addressing architectyre divides all possible IP addresses into classes based on the nost significant bits of the address. The three main address classessare ClassA ,Class B,and Class C.

Class A Addresses have a binary zero(0) ass tge nist significant bit of the address. The address range is from 0 0.0.0.0 to 126.255.255.255.

The network portion is the most significant byte of the addresses and the host portion is the three byte remainder.Class A network addresses are inteneded for networks that have more than 65,534 hosts on as single network

Class B Addresses range is from 128.0.0.0 to 191.255.255.255 . The network portion is the most significant two bytes of the address and the host portion is the two byte remainder.Class B network addresses are intended for networks that have from 255 to 65,534 host on single network

Class C addresses have binary One One Zero as the most Significant three bits of address.The address range is from 192.0.0.0 to 223.255.255.255. The network portion is the most significant three bytes of address and the host is one byte remainder

Subnetting a Network

Subnet masks can divide networks into smaller network.In order to understand this process, a discussion on binary-to-decimal number conversion is required.

The typical subnet mask 255.255.255.0 represents 4 byte of data.Each number represent 1 byte and is displayed as a decimal numbers.One byte consists of 8 bits,where 0000 0000 represents 0(in decimal) and 1111 1111 represents 255(in decimal)
