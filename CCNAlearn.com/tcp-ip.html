TCP/IP MOdel

TCP/IP is acroynm is Tramission Control Protocal and Internet Protocol.It is the network model used in the current Internet architecture as well. Protocols are set of rules which govern every possible communication over a network. These protocols describe the movement of data between the source and destination or the internet.

  1. Host-to-network layer:
    • Used in connection to host.
    • This is lowest layer of Tcp/ip model
  2. Internet layer:
    • Its is independently packet are reach to destionation
    • Internet protocol are used in this layer.
    • Its is connectionless internetwork layer.
  3. Transport Layer:
    • It decided the data is tranfer in which form and also decides the path.
    • This layer adds header information to the data.
    • This layer also arrange the packets in serial/sequence vice.
  4. Application Layer:
    1. Telnet:Its enables users on one machine to login into other computer on the net telnet can be beneficial because is enables you server or host computer without sitting at its Keyboard.
    2. FTP: is protocol ,that allows the file transfer those whose connected over a network.
    3. SMTP(Simple Transport Protocol) is a protocol, which used to transport electronic mail between source and destination.
    4. DNS(Domain Name Server) resolves an IP address into a textual address for Hosts connected over a network
