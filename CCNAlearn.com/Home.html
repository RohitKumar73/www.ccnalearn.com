What is Cisco Packet tracer

Ans:A packet tracer is a simulation software thar can be used to simulate the behaviour of real networking equipments like a router , a hub or a switch, a WAN cloud, a PC/server/printer and a wireless access ;point or IP telephone.Its allows users to create network topologies, configures devices,inject packets and simulate the netord in multiple visual representations

Uses of packet tracer

  1. Design,build,configure and trouble shoot networks using virtual equipments.
  2. Demostrate technologies and configures.
  3. Supplement classroom equipment.
  4. To empower students to explore concepts,conduct experiments and test their understanding.

Features of packet tracer

  1. Real-time simulation modes.
  2. Logical topology and physical modes.
  3. Global packet sniffer called as event viewer.
  4. RIP(Version 1,Versoin 2),ICMP,ARP,DHCP.
  5. Ethernet,VLANs, PPP,HDLC.
